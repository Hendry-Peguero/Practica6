Servidor NginX con Docker

¿Qué es esta implementación?

Este proyecto demuestra cómo configurar y ejecutar un servidor web utilizando NginX dentro de un contenedor de Docker. Utilizando Docker Compose, levantamos el contenedor de manera automática para servir esta página.

Objetivos de la práctica

  • Crear un entorno de desarrollo estable y replicable.
  • Configurar un servidor web NginX en un contenedor Docker.
  • Usar Docker Compose para simplificar la configuración del contenedor.
  • Versionar el proyecto en GitHub.

Gracias a Docker, esta configuración es portable y fácil de desplegar en distintos entornos. Esto permite replicar el servidor web en cualquier máquina con Docker instalado.
